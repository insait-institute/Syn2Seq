## Overview

This repository contains the official implementation of **"From Synchrony to Sequence: Exo-to-Ego Generation via Interpolation"**, accepted at **ECCV 2026**.

We introduce a sequential formulation for exo-to-ego video generation that bridges the synchronization-induced discontinuity between exocentric and egocentric views through interpolation, enabling more coherent cross-view video synthesis.
